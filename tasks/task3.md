# Task 3

## Description

Given an expression string *exp* , write a program to examine whether the pairs and the orders of “{“,”}”,”(“,”)”,”[“,”]” are correct in *exp* (true), or incorrect (false).

Example:

Input:
> <tt>“[()]{}{[()()]()}”</tt>

Output:
> <tt>true</tt>

Input:
> <tt>“[(])”</tt>

Output:
> <tt>false</tt>
